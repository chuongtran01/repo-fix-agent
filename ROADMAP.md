# Roadmap

## Phase 1: Local Repo MVP

- File tools
- Search tools
- Basic LangGraph workflow
- One test command
- Final summary

## Phase 2: Safer Editing

- Git diff support
- Rollback support
- Max file change limits
- Better command allowlist

## Phase 3: Smarter Debugging

- Parse test failures
- Route back to relevant files
- Retry with explanation
- Track failed attempts

## Phase 4: Developer Workflow

- Generate patch files
- Create Git branches
- Open pull requests
- Add human approval before final changes
